# YangTu

**YangTu** is a service platform that enables users to order home cleaning and shopping delivery services in a single app. Users can choose to be **Jalu** (service requester) or **Yusa** (task performer). This app provides flexibility and convenience for both parties, creating an efficient and user-friendly experience for daily needs.

## Key Features (MVP)

1. **Daily Task Assistance**: Users can request assistance for a variety of daily tasks such as cleaning houses, cars, pools, or even shopping for items.
2. **Chat Feature**: Allows direct communication between Jalu (job requester) and Yusa (service provider). This feature helps ensure seamless coordination until the task is completed.
3. **Google Maps Integration**: A map-based delivery feature to ensure timely and accurate delivery services.

## Tech Stack

**Frontend**:
- **React**: For the web application.
- **React Native**: For the mobile application.
- **Tailwind CSS**: For responsive and efficient styling.
- **Redux**: For global state management.
- **Socket.io**: For real-time communication.
- **DaisyUI**: UI components integrated with Tailwind.

**Backend**:
- **Express**: Server-side framework for Node.js.
- **MongoDB**: NoSQL database for data storage.
- **Mongoose**: ODM to facilitate MongoDB interaction.
- **Midtrans**: Payment integration for secure transactions.
- **Google Maps API**: For map-based delivery features.
- **Redis**: Cache to boost application performance.
- **Axios**: Library for HTTP requests.
